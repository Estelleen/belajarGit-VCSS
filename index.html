Github adalah sebuah platform yang memungkinkan developer untuk membuat, menyimpan, mengatur serta membagikan kode yang dibuat. Github menggunakan git untuk menyediakan distributed version control dan github sendiri menyediakan kontrol akses, tracking bug, software feature request, task management, continues integration serta wikis untuk setiap proyeknya.

Dalam GitHub terdapat istilah repository atau yang sering disebut sebagai repo dan berbagai macam reponya, yaitu: Local Repo dan Remote Repo. Selain itu juga ada yang namanya Push yang berarti mengirimkan perubahan dari lokal ke remote.
